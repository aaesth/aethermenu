# aethermenu
paper plugin to hide players and show a menu for hub servers

im also making this to learn java so my code is ass

example config
```
menu:
  title: "§8choose a server"
  size: 3

  freeze-player: true
  hide-players: true

  items:
    survival:
      material: DIAMOND_SWORD
      name: "§asurvival"
      slot: 1
      server: "survival"

    creative:
      material: GRASS_BLOCK
      name: "§bcreative"
      slot: 3
      server: "creative"
```
